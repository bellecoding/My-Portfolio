Case Study

I developed a custom AI assistant designed to guide customers through complex product configuration and quotation processes for an international e-commerce company specializing in technical systems.

Role & Stack

  • Role: Lead Developer & AI Engineer
  • Stack: React, TypeScript, Node.js, OpenAI API, PDFKit, Supabase, Magento API

Project Overview

The assistant functions as an intelligent product advisor that interacts with users through a conversational interface. It collects key installation parameters (measurements, environment type, and product preferences), applies reasoning logic to determine compatible components, and automatically generates detailed quotations in both PDF and JSON formats.

On the backend, I built a Node.js API that connects the assistant with the companyâ€™s ERP and e-commerce infrastructure, enabling the system to send structured product data, generate quotes dynamically, and deliver AI-written emails with attached PDFs directly to customers. The solution includes a conversation history database, quote tracking, and automatic Magento SKU integration.

Key Features

  • Guided conversational flow powered by GPT-4-Turbo for step-by-step system configuration.
  • Automated quotation builder generating PDFs and JSON payloads for backend processing.
  • Magento API integration for SKU tracking, add-to-cart links, and email automation.
  • Real-time knowledge base linking, embedding relevant articles and resources.
  • Vision prototype allowing users to upload photos or sketches for AI-assisted form completion.
  • Frontend chat widget in React/TypeScript, auto-updated via Netlify with a single HTML snippet.

Impact

This project transformed a manual quoting workflow into a scalable, automated solution that operates across multiple regional websites, reducing support workload by over 60% and enabling instant, AI-driven customer responses 24/7.
